# Code Challenge
## Simple app bank 

Android application build with kotlin 

## Features

- Consume Firebase firestore to save and get data
- You can register a new user requesting personal information
- With the registered credentials you can login
- In the home of the application you can see the user's name, their balance and the list of movements
- You can see the detail of each movement


## Tech

Used libraries and technologies

- [FIREBASE] - for rest api consumption
- [DAGGER] - for dependency injection
- [MVVM] - MVVM architecture 
- [VIEW BINDING] - to access the graphic elements from the class
